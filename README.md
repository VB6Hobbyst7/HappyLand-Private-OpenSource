# HappyLand Private
HappyLand Private (OpenSource) ที่ผมเคยทำแจก , โดเนท โดยการใช้ WKE , YDark หรือโปรแกรมอื่นๆ บลาๆ เพื่อทำการบายพาส EasyAntiCheat เพื่อใช้บอทเกมส์ Ragnarok โดยการทำงานคร่าวๆคือการ ทับไฟล์ EasyAntiCheat เวอร์ชั่นเก่าจาก Resources ภายในตัวโปรแกรมลงไปในตำแหน่งของ EasyAntiCheat ภายในเครื่อง เพื่อทำให้สามารถมองเห็น Threads , Callback , Sys , module ต่างๆของ EasyAntiCheat เพื่อให้สามารถบายพาสได้

# Source Code
ภายใน Source Code นี้ จะมี comment อธิบายฟังก์ชั่นต่างๆ อย่างเช่น Replace File (ทับไฟล์) , Verify File (ยืนยันไฟล์ และทำการบล็อก firewall) , Start RO (เริ่มเกม Ragnarok โดยใช้ EasyAntiCheat.exe) , Reset (Reset firewall ต่างๆเพื่อให้กลับเป็นค่าเดิม) 

# dev note
จริงๆผมจะปล่อย source code ตั้งนานแล้วแหล่ะ แต่ไม่มีเวลาว่างมาปล่อยเลย ตอนนี้ว่างก็เลยมาปล่อยให้ *อย่าคาดหวังอะไรจากโค๊ดและโปรแกรมขยะๆนี้เลย* อาจจะไม่ได้โค๊ดสวยหรือ clean มากขนาดนั้นแต่มันก็ใช้ได้ (จริงมั้ย?! 555+)
เป็นโปรเจ็คที่ผมเคยทำเพื่อแจกใน Discord แฮปปี้แลนด์เพื่อให้ทุกคนได้ใช้กัน แต่ตอนนี้มัน outdate ไปแล้วไม่สามารถใช้ได้ (เอ๊ะหรือว่าได้? หาวิธีกันเอาเอง) จุดประสงค์ที่ปล่อยให้เป็น opensource ก็คือ เผื่ออาจจะมีคนที่อยากทำอะไรแบบนี้แล้วไม่รู้จะหาแหล่งข้อมูลหรือตัวอย่างจากไหนก็เอาจากนี้แหละเป็นตัวอย่าง โค๊ดอาจจะไม่สวยนัก แต่โดยรวมแล้วการทำงานหลักๆก็ถือว่าโอเค เพราะมันไม่ได้มีอะไรที่มันซับซ้อน เผลอๆเอาไปทำต่ออาจจะทำได้ดีกว่านี้อีกก็ได้
